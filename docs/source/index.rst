Welcome to Clypher's documentation!
===================================

**Clypher** is a small Python program that allows you to encrypt and decrypt files and folders via a simple drag and drop interface.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

.. note::
   
   This project is intended as a little personal project, and not as an actual secure solution for file encryption.
   If you're looking for a secure solution, use other utilities.
